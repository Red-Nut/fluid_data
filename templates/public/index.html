Fluid Data

Welcome to Fluid Data, the number one location for downhole data in Australia.

Overview

Fluid Data is a collection of openfile well data provided in an easily accessible database. In the fluid data database, the metadata, reports and files for each well have been linked together to form one accurate source of information. Additionally, the reports have been analysed with machine learning to extract buried data.

Data Set

The dataset includes:

  • Well Coordinates
  • Permit Number
  • State
  • The Well Purpose (e.g. CSG, water, CO2 sequestration)
  • The Well Class (e.g. Development, Production, Pilot)
  • The Well Operator
  • The Rig Release Date
